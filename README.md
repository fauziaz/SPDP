# SPDP - Sistem Pendukung Diagnosis Penyakit

SPDP (Sistem Pendukung Diagnosis Penyakit) adalah aplikasi berbasis web yang dirancang untuk membantu proses diagnosis penyakit menggunakan metode *rule-based system* dan algoritma klasifikasi **C4.5**.

---

## Fitur Utama

- Diagnosis penyakit berbasis gejala
- Rule-based system (basis pengetahuan)
- Dashboard interaktif
- Integrasi database

---

## Teknologi yang Digunakan

- Python
- Django
- HTML, CSS, JavaScript
- SQLite
- Machine Learning (C4.5 Algorithm)
